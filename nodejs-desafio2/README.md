# APP

A app for diet control daily

# RFs (Functional Requirements)

- [ ] Must be possible to create a user;
- [ ] Must be possible to identify a user within requests;
- [ ] Must be possible to register a meal, with the following attributes:
        - Name
        - Description
        - Date and Hour
        - Whether it is in or out the diet (diet status)
- [ ] Must be possible to edit a meal, all attributes can be edited.
- [ ] Must be possible to delete a meal
- [ ] Must be possible to view a meal
- [ ] Must be possible to track user metrics
    - Total of registered meals
    - Total of meals in the diet
    - Total of meals out the diet
    - Best sequence of meals in the diet
- [ ] The user can only view, edit and delete their own meals (those they have created)
